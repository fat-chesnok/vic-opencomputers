# Vic
Vic (Vi Cubed) is a text editor for Minecraft mod "Open Computers"
operating system OpenOS inspired by Vim. 

User Interface is planned to be more similar to configured Neovim, than 
to original Vi, but in future I guess I can make Vic more configurable
and give user possibility to choice which interface he wants! 

Currently in not so active development but maybe soon...

# Screenshots
You can configure color theme by editing configuration file `/etc/vic/init.vic`.
Stock colors are:
```Lua
vic.BG_COLOR = 0x1E1E1E
vic.LINE_COLOR = 0x4B4B4B
```
